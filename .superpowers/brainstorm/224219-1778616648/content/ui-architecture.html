C. Wizard / linear workflow

Forced step-by-step flow: Import → Transcribe → Tag → Analyze → Export. Simpler if researcher always works linearly, but real QDA work cycles back constantly (re-code, refine codebook, re-read earlier interviews). Likely too rigid for her.
